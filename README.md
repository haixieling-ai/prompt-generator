# Prompt Generator

一个基于 **Next.js + TypeScript + Tailwind CSS** 的产品视觉提示词生成器。

## 功能

- 九维输入参数（产品类型、风格、场景、镜头角度、光线、材质、构图、摆件、约束条件）
- 自动生成中文/英文提示词
- 一键复制中文或英文结果
- localStorage 本地历史记录（最多保留 8 条）
- 简洁高级的深色界面

## 本地运行

```bash
npm install
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000)。
